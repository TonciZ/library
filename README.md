 ##Movie
Properties:

- director (string)

- runTime (number)
- isCheckedOut (boolean, initially false)
- ratings (array, initially empty)

Getters:

- all properties have a getter

Methods:

- `.getAverageRating()`
- `.toggleCheckOutStatus()`
- `.addRating()`

 ##CD
Properties:

- artist (string)

Getters:

- `.addRating()`
